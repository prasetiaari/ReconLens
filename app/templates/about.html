About ReconLens

ReconLens was created by Prasetia Ari, with continuous support from ChatGPT-5 throughout the development process. The tool is inspired by gau-expose, with the original idea of building a more user-friendly UI version so reconnaissance work is not limited to CLI or scripts.

Development started while participating in the Tangsel Bug Bounty 2025 program. Because of that, many features in ReconLens are directly derived from real bug hunting needs and have been tested in actual bounty scenarios.

Our Goals

  • Simplify reconnaissance workflows — so security researchers don’t need to constantly juggle multiple scripts manually.
  • Provide a practical dashboard — to visualize and analyze recon results more effectively.
  • Pave the way for AI integration — early experiments have already been implemented, and further development will continue.
